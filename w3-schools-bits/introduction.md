# Introduction

## What is Go
Go is a simple programming language developed at Google by  by Robert Griesemer, Rob Pike, and Ken Thompson in 2007, Go syntax is similar to C++

# Go program structure

### Every Go program has these parts:

- Package :defines the program (usually main)
- Imports :brings in extra tools, like fmt for printing
- Functions:where your code runs, e.g., func main()
- Statements :instructions the computer executes, e.g. fmt.Println("Hello World!")

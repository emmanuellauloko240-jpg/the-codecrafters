# Loops

A for loop iterates over a block of code a specified number of times.

# the continue statement
the continue statement is used to ignore step in the loop and continue with the next step in the loop.

# the break statement
the break statement is used to quit the loop process.

# Nested Loop
this simply mean adding a loop inside another loop

# The Range Keyword
the Range Keyword is used to loop through the list of an array, slice, map and return both the index and the value.

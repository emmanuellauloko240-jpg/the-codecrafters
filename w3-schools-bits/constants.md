# constants
a constant is a fixed value that cannot be changed.

# Constant rules
- constant cannot start with digit
- Constants can be declared inside a function or outside a function
- Once a value is assigned to a constant, it cannot be changed throughout the program.
# Constant Types

## There are two types of constants:

 - Typed constants: typed constants are declared with a defined type

 While

 - Untyped constants :untyped constants  are declared without a type

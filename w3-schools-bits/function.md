# Go functions
A function is a reusable block of code that performs a specific task.

# Create a Function
Use the func keyword.
Give the function a name followed by ().
Write the code inside {}.

# Calling a Function
Functions are executed by calling their name with parentheses, functionName().
A function can be called multiple times.

# Naming Rules for Go Functions
function name must start with a letter, contain alpha-numeric characters and underscores and can`t contain spaces.

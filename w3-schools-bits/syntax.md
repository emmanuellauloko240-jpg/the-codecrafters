# Go syntax
A Go file consists of the following parts:
- package declaration
- import packages
- functions
- expressions 

## Explanation

- The package declaration is described using the package keyword. 

- import ("fmt") allows us import files included in the fmt package

- A black line is an empty  line. Go doesn’t care about spaces but spaces make the code easier to read.

- func main() is where the program starts, and anything inside the  curly brackets {} will be executed.

- fmt.Println() is a function from the fmt package that displays text on the screen by printing "Hello World!" as  output.

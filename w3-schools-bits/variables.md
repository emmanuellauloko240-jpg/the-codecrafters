# Variables 

Variables are container used for storing Data values

# Types of variables in GO
- int: Takes in intergers
- strings: strings takes in word like "how are you" with double quotes closing it
- float: Takes in floating numbers with decimal
- bool: Takes in values in two states which is either True/False

# Declaring variables in GO
variables are declared in two ways using:

- The var keyword and this sign (:=)
## When using the Var keyword it is followed by the variable name and Type 
while when using this sign (:=) it is followed by the variable value

# Difference betwee Var and :=
 
 - var is used inside and outside the function 
 while
 - := is used only inside function.

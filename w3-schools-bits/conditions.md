# Condition

conditional statement lets the program make decisions based on whether a condition is true or false.

# Go supports comparisonn operators from mathematics like:
- Less than <
- Less than or equal <=
- Greater than >
- Greater than or equal >=
- Equal to ==
- Not equal to !=

# Go also supports the usual logical operators such as:
- Logical AND &&
- Logical OR ||
- Logical NOT !

# Go has the following conditional statements:
- the "If" Execute a block of code only if a condition is true.
- The "else" Execute a block of code if the if condition is false.
- the "else if" Test a new condition if the previous if (or else if) condition was false.
- the "switch" Choose one block to execute among many alternatives based on a value.
